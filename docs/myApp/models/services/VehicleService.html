Class VehicleService

lucee.Component
    myApp.models.services.VehicleService

Wraps the Vehicles API to provide higher-order methods for pulling vehicle data. If you find yourself manipulating vehicle data frequently and consistently, consider moving that vehicle transforming logic into this component.
